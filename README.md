# LVGL Display - ESPHome for Home Assistant
Home Assistant Dashboard on ESP32-S3 LVGL display using ESPHome.

This code is designed to run on an LVGL Display via ESPHome for Home Assistant. 

![20251112_crowpanel](https://github.com/user-attachments/assets/31cc059a-17da-452f-a60c-0ead8bb9a6be)

I am using a 5.0' model from Elecrow which is not the latest version as of November 2025 (there is now an AI model). This particular model is the ESP32-S3 standard model and is performing well.
The white bezel was 3d printed from the original source

https://www.elecrow.com/display/esp-hmi-display/esp32-hmi-display.html

https://www.elecrow.com/wiki/esp32-display-502727-intelligent-touch-screen-wi-fi26ble-800480-hmi-display.html

Code was developed using many of the examples available on esphome.io , lots of trial and error and claude.ai

https://esphome.io/cookbook/lvgl/
